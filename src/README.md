# Source Files

This directory contains the source code for SnakeClaude.

## File Structure

- `main.js` - Entry point of the application, initializes the game
- `config.js` - Game configuration constants (grid size, speed, colors, etc.)
- `game.js` - Core game logic and state management
- `renderer.js` - Canvas rendering logic
- `input.js` - Keyboard and touch input handling
- `ui.js` - UI elements management (score, etc.)
- `network.js` - Multiplayer networking via Supabase
- `styles.css` - Game styling

## Testing

Each module should have a corresponding `.test.js` file for unit tests following TDD principles.
